## 1. Общее описание проекта

**MyBlog** — это веб-приложение для управления блогом, предоставляющее пользователям инструменты для создания, редактирования и публикации статей в интернете. Проект разработан на основе фреймворка Flask с использованием SQLAlchemy для работы с базой данных и TinyMCE для редактирования контента.

---

## 2. Цель проекта

Целью разработки MyBlog является создание полнофункционального веб-приложения, которое позволит:
- Пользователям регистрироваться и управлять своими учетными записями
- Создавать и редактировать блог-посты с использованием удобного визуального редактора
- Просматривать опубликованный контент других пользователей
- Администраторам управлять содержимым и пользователями системы

### 3.1 Аутентификация и управление пользователями
- Система регистрации новых пользователей

- Создание новых блог-постов

### 3.3 Навигация и поиск

- Организованный пользовательский интерфейс

- Управление контентом (одобрение, удаление постов)
